## Featured Project — Vision-Based SHM System

Developed a low-cost structural monitoring system using ESP32-CAM and computer vision to measure displacement and strain with micrometer-level sensitivity.

### Key Contributions
- Image-based displacement tracking using computer vision  
- Subpixel accuracy for micro-displacement detection  
- Embedded data acquisition using ESP32-CAM  
- Experimental validation using LVDT reference system  

### Why it Matters
Traditional SHM systems are expensive and difficult to scale. This project demonstrates a practical, low-cost alternative for infrastructure monitoring.
